# Precio piloto

## Objetivo

Definir una oferta inicial atractiva para conseguir los primeros casos de éxito.

---

# Oferta piloto

## Incluye

- Landing simple o integración con página existente.
- Chatbot FAQ.
- Captura de leads.
- Agendamiento básico.
- Recordatorios.
- Dashboard básico.
- Soporte durante el piloto.

---

# Precio inicial sugerido

## Opción 1: Entrada baja

Setup: $400  
Mensualidad: $50 - $100

## Opción 2: Más seria

Setup: $700 - $1000  
Mensualidad: $100 - $150

## Opción 3: Caso de estudio

Setup reducido: $300 - $500  
Mensualidad: $50 - $100  
A cambio de testimonio y permiso para usar resultados como caso de éxito.

---

# Recomendación

Para el primer cliente, ofrecería:

Setup: $500 - $700  
Mensualidad: $100  

Condición:

- Se usa como caso de estudio.
- Se mide número de leads y citas.
- Se pide testimonio si funciona bien.

# Definition of Done

## Objetivo

Definir cuándo la demo se considera realmente lista para mostrar a una clínica dental.

---

# La demo está lista cuando:

1. La landing page se puede abrir correctamente.
2. El usuario puede iniciar una consulta.
3. El chatbot responde una pregunta frecuente.
4. El sistema identifica el tratamiento de interés.
5. El paciente puede elegir una fecha y hora.
6. La cita se guarda en la base de datos.
7. El lead se guarda en la base de datos.
8. El sistema envía confirmación.
9. El dashboard muestra la cita agendada.
10. El flujo completo puede demostrarse en menos de 5 minutos.

---

# No se considera lista si:

- Hay que explicar demasiado para que se entienda.
- La cita no se guarda.
- El chatbot responde de forma insegura.
- No se puede ver el resultado en dashboard.
- Solo funciona si alguien del equipo interviene manualmente.
- No se entiende qué problema resuelve.
- No se puede mostrar a una clínica sin improvisar demasiado.

---

# Criterio final

La demo debe permitir decirle a una clínica:

"Escriba como si fuera un paciente interesado en brackets."

Y el sistema debe poder:

- responder
- clasificar la necesidad
- ofrecer agendar
- guardar datos
- confirmar la cita
- mostrar el resultado en dashboard

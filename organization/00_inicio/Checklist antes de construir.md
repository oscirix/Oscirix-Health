# Checklist antes de construir

## Producto

- [ ] Definir flujo exacto del paciente.
- [ ] Definir funcionalidades del MVP.
- [ ] Definir qué NO se va a construir.
- [ ] Definir demo exacta.

## Tecnología

- [ ] Confirmar arquitectura.
- [ ] Crear tablas en Supabase.
- [ ] Definir APIs necesarias.
- [ ] Definir flujo n8n.
- [ ] Confirmar Twilio o alternativa.

## Branding

- [ ] Nombre temporal.
- [ ] Logo provisional.
- [ ] Colores provisionales.
- [ ] Mensaje principal.
- [ ] Pitch de 30 segundos.

## Ventas

- [ ] Lista de 10 clínicas.
- [ ] Guion de contacto.
- [ ] Precio piloto.
- [ ] Presentación corta.
- [ ] Simulación de venta.

## Demo

- [ ] Landing visible.
- [ ] Chatbot responde.
- [ ] Paciente agenda.
- [ ] Datos se guardan.
- [ ] Dashboard muestra la cita.

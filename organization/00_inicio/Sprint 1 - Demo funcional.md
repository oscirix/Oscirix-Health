# Sprint 1 - Demo funcional

## Objetivo

Construir una demo funcional en máximo 1-2 semanas para presentar a una clínica dental.

---

# Resultado esperado

La demo debe permitir:

1. Entrar a una landing.
2. Hacer una consulta.
3. Recibir respuesta tipo FAQ.
4. Agendar una cita.
5. Guardar paciente y cita.
6. Ver datos en dashboard.
7. Enviar confirmación o recordatorio.

# Criterio de éxito

La demo está lista cuando podamos decirle a una clínica:

"Escriba aquí como si fuera un paciente."

Y el sistema pueda:

- responder
- clasificar necesidad
- agendar
- guardar datos
- mostrar resultado

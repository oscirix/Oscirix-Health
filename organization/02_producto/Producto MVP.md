# Producto MVP

## Objetivo

Crear una demo funcional para presentar a una clínica dental.

## Funciones principales

1. Landing page
2. Chatbot FAQ
3. Captura de leads
4. Agendamiento
5. Reagendamiento
6. Recordatorios
7. Dashboard básico

## Flujo principal

Paciente escribe por WhatsApp o landing.

↓

Sistema pregunta qué necesita.

↓

Paciente selecciona tratamiento.

↓

Sistema responde información básica.

↓

Sistema ofrece agendar cita.

↓

Paciente elige fecha y hora.

↓

Sistema guarda datos.

↓

Sistema envía confirmación.

↓

Sistema envía recordatorio.

## No incluye por ahora

- Historia clínica
- Diagnóstico
- Odontograma
- Recetas
- Facturación
- Pagos
- Radiografías
